界面和你现在的 demo 一样 —— 区别全在背后：房间里的人和数字都是你们 App 的真实用户数据，不是写死的假人。下面把"真实"标出来给你看。

背后的数据（同一块屏幕的真相）

① 在线人数 = 一句 SQL
-- 最近15分钟有专注记录的真实用户数
select count(distinct user_id)
from study_sessions
where started_at > now() - interval '15 min';
② 房间里的人 = 真实用户快照
-- 取最近在学的真实用户，读他们的资料
select name, country, avatar_cfg,
  streak, total_hours, current_task, goal
from profiles join study_sessions ...
order by last_active desc limit 6;
③ 你专注结束 = 写一条记录
insert into study_sessions
  (user_id, scene, minutes, started_at)
values ('me', '图书馆', 25, now());
